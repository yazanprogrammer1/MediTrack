## Tools and Libraries Used

- Language: Kotlin
- Framework: Android Jetpack Compose
- Image Loading: Glide
- Dependency Injection: Hilt
- Asynchronous Programming: Coroutines, Flow
- Android Architecture Components: MVVM, ViewModel
- Backend: Firebase
